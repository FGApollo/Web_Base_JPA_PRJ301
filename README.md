# Web Basic Project using JPA (Backend Focus)

This is a learning project for practicing **Java Web Development** using:
- Java Servlets & JSP
- JPA (Java Persistence API)
- MVC architecture
- SQL Server (via JDBC driver)

## Features
- User login / logout
- Create new account
- Update account
- Delete account
- Search users by last name
